Public Bathrooms Audit — San Diego County

An open-source geospatial project auditing public bathroom access across San Diego County to support accessibility planning, service delivery, and policy decision-making.

Project overview

  • Goal: Create a reliable, auditable inventory of public bathroom access
  • Unit of analysis: Individual bathroom locations (points)
  • Use cases: Public access, service planning, policy analysis
  • Status: In progress — migrating from proprietary GIS to open source

Engineering focus

  • Schema-driven validation for user-submitted locations
  • Separation of ingest, validation, storage, and visualization layers
  • Support for iterative updates and QA workflows
  • Designed to scale beyond a single city or dataset

Current implementation (legacy)

This ArcGIS Online map represents the current operational version of the project. It is included here for reference while the open-source pipeline is under active development.

  • Proprietary hosting (ArcGIS Online)
  • Manual update workflows
  • No automated validation or versioning

Planned replacement: open-source web map backed by FastAPI ingestion, schema validation, and a spatial database.

Planned migration roadmap

  • Define and publish a public bathroom schema
  • Build FastAPI endpoints for data submission and review
  • Store validated data in PostGIS
  • Replace proprietary map with open-source web visualization
