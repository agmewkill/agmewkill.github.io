Public Bathroom Access Audit ‚Äî San Diego County

An open-source geospatial project auditing public bathroom access across San Diego County. This work transitions an existing ArcGIS Online mapping effort into a reproducible, pipeline-driven data product that supports both policy analysis and engineering workflows.

Why this project

  • Public bathroom access is a critical public health and accessibility issue
  • Existing data is often fragmented, outdated, or locked behind proprietary platforms
  • This project prioritizes transparency, auditability, and open data principles

Current state

  • Bathroom locations and attributes currently exist in ArcGIS Online
  • Attributes include access hours, ownership, and accessibility indicators
  • This repository documents the transition to an open-source workflow

Engineering focus

  • Point-based geospatial data ingestion and normalization
  • Schema definition and validation
  • Repeatable ETL pipelines for updates and QA
  • Preparation for user-submitted data and moderation workflows

Policy relevance

  • Homelessness response and service access planning
  • ADA accessibility audits
  • Public infrastructure gap analysis
  • Evidence-based advocacy and planning

Planned next steps

  • Export ArcGIS Online data to open formats (CSV / GeoJSON)
  • Define and version dataset schema
  • Implement validation and QA pipelines
  • Publish a public-facing interactive map
