Homeless Encampments & Social Determinants of Health

This project analyzes the relationship between tract-level homeless encampment presence and social determinants of health in San Diego County using privacy-preserving, census-tract–aggregated data.

Key design principles

  • Privacy-first: Encampments are already aggregated by census tract — no point locations
  • Unit of analysis: Census tract (GEOID)
  • Data sources: ACS 5-year estimates + tract-level encampment counts
  • Purpose: Produce analysis-ready data for descriptive statistics and modeling

Workflow overview

  • Ingest ACS 5-year tract indicators (demographic, economic, housing)
  • Clean and normalize variables for comparability
  • Join encampment counts to ACS data using tract identifiers
  • Generate tract-level tables for downstream analysis and visualization
