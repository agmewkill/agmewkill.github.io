PITC × ACS Correlations (San Diego County, 2013–2025)

This page documents the methods, assumptions, and analytical context for the PITC × ACS Correlations project. The interactive map and correlation panel are the second half of this same project.

Launch interactive View code

What this project demonstrates

  • Multi-year data ingestion using year-configured CSV inputs and shared tract geometry
  • Schema normalization for real-world ACS column inconsistencies
  • Geospatial joins between tract-level PIT counts and demographic indicators
  • Inline QA and interpretability features (hover details, legends, correlation summaries)
  • Statistical computation of Pearson correlation (r) with safeguards for missing data

Data & assumptions

  • Inputs include PIT count totals by census tract and ACS 5-year tract-level indicators
  • Tract geometry is simplified for performance and interactive use
  • Some PIT years reuse the most recent available ACS vintage
  • Correlation values describe linear association only and do not imply causation

Pipeline alignment

This project represents the downstream visualization and QA layer of a typical geospatial ETL workflow: ingest → transform → validate → output. Related pipeline structure and scripts can be found in pipelines/geospatial-data-pipelines.
