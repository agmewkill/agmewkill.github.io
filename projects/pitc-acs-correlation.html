PITC × ACS Correlations (San Diego County, 2013–2025)

Interactive p5.js map for exploring tract-level PIT count totals and Pearson correlation (r) with ACS demographic indicators across multiple years.

Launch interactive View code

What this demonstrates

  • Multi-year data ingestion pattern (year-configured CSVs + shared tract geometry)
  • Schema normalization for messy real-world headers (candidate column matching)
  • Geospatial join strategy (tract IDs + pre-simplified GeoJSON geometry)
  • On-the-fly QA and interpretability features (hover details, legend, correlations panel)
  • Statistical computation (Pearson r) with safeguards for missing data and low sample size

Data & assumptions

  • Inputs: PIT count totals by tract (selected years) + ACS 5-year tract indicators
  • Tract geometry: simplified San Diego County tract boundaries (GeoJSON)
  • Some PIT years may share the same ACS vintage if the newest ACS release is not available
  • Correlation is descriptive (linear association), not causal

Pipeline alignment

This visualization sits at the “downstream” end of a typical geospatial ETL workflow: ingest → transform → validate → output. Related pipeline structure: pipelines/geospatial-data-pipelines.
